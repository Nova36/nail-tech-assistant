Spec.

Three entries. Dashboard (idle), New Design (with subtle "New" pip — auto-removed after first click), Library (no pip). Order: top-down by frequency expected — Dashboard for at-a-glance, New Design for primary action, Library for review.
Active treatment. 3px primary rail on the left of the row + bg-primary/12 tint + primary-color text + 500 weight + active icon recolor. Not color-only — the rail + weight reinforce.
Hover/idle. Idle = transparent bg, foreground text, muted-foreground icon. Hover = rgba(140,126,120,0.08) bg, no rail. Min hit target 44px.
Pinterest absence. Intentional. The picker is reachable at /pinterest by URL, and is embedded as a panel inside /design/new. Sidebar avoids the implication that Pinterest is a destination.
Header + footer. Logo block unchanged. User chip (avatar + name + Sign out) unchanged. Only the nav middle section changes.
A11y. nav aria-label="Primary". Active row carries aria-current="page". "New" pip uses aria-label so SR reads "New Design — new feature". Tab order = Dashboard → New Design → Library → user chip → Sign out.
